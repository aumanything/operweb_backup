About AWS Cloud

November 6, 2024 Chartwit

What is Troubleshooting?

Amazon Web Services (AWS) คือแพลตฟอร์มคลาวด์ที่ให้บริการหลายรูปแบบ เช่น การประมวลผล การเก็บข้อมูล เครือข่าย และฐานข้อมูล AWS ช่วยให้องค์กรทุกขนาดสามารถสร้างแอปพลิเคชัน ปรับขนาดระบบได้อย่างยืดหยุ่น และเพลิดเพลินกับความปลอดภัยระดับสูง


โมเดลไคลเอนต์-เซิร์ฟเวอร์คืออะไร

Client-Server Model เป็นรูปแบบการสื่อสารระหว่างอุปกรณ์สองชนิดคือไคลเอนต์และเซิร์ฟเวอร์ โดยไคลเอนต์ทำการร้องขอข้อมูลหรือบริการจากเซิร์ฟเวอร์ที่เป็นผู้ให้บริการ เช่น เว็บเซิร์ฟเวอร์หรือฐานข้อมูล เซิร์ฟเวอร์จะตอบสนองต่อคำขอนั้น โมเดลนี้ใช้กันมากในแอปพลิเคชันบนเว็บและบริการออนไลน์ต่าง ๆ

โมเดลการปรับใช้งานการประมวลผลบนระบบคลาวด์ Cloud Deployment Models ประกอบไปด้วยรูปแบบหลัก 3 แบบ:
1. Public Cloud: เป็นระบบคลาวด์ที่บริการเปิดให้สาธารณะ โดยผู้ให้บริการ เช่น AWS, Azure ให้บริการทรัพยากรที่โฮสต์ในศูนย์ข้อมูลของพวกเขา
2. Private Cloud: เป็นระบบคลาวด์ที่สงวนไว้สำหรับการใช้งานภายในองค์กรเท่านั้น มีความปลอดภัยสูงและควบคุมได้ง่าย
3. Hybrid Cloud: ผสมผสานระหว่าง Public และ Private Cloud โดยเลือกเก็บข้อมูลที่ต้องการความปลอดภัยสูงใน Private Cloud ส่วนข้อมูลอื่น ๆ จะเก็บใน Public Cloud
